Regional Sales Pipeline Dashboard – README

Project Overview

This project presents a complete Regional Sales Pipeline Analysis Dashboard for the year 2025.
The dashboard helps identify sales bottlenecks, cancellation patterns, revenue loss areas, and top-performing regions using business intelligence concepts.

The project was designed in a Power BI style interactive dashboard using HTML, CSS, JavaScript, and Chart.js.

Objective

The main goal of this project is to analyze:

Regional sales performance
Revenue trends
Cancellation & return rates
Product category contribution
Top-performing sales agents
Revenue loss by product and region

The analysis helps management take data-driven business decisions.
