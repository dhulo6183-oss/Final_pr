# 📊 Excel Analyzer PR.2

> A complete Excel-based Sales Performance Analysis project using Pivot Tables, Conditional Formatting, What-If Analysis, Regression Analysis, Descriptive Statistics, and Interactive Charts.

---

# 🚀 Project Overview

This project analyzes sales data using advanced Microsoft Excel features and data analysis techniques.  
The objective of this project is to transform raw sales data into meaningful business insights through dashboards, visualizations, and statistical analysis.

---

- Regression Analysis
- Descriptive Statistics
- Charts & Dashboarding
- Excel Formulas
  - `IF()`

  - `MATCH()`
  - `NOW()`

# 📌 Features Implemented

## ✅ Conditional Formatting
- Highlighted Top 10 customers based on total purchase amount.

## ✅ What-If Analysis
- Analyzed impact of changing discount percentage on profit.

## ✅ Linear Regression Analysis
- Performed regression analysis using Excel Data Analysis ToolPak.

## ✅ Descriptive Statistics
- Generated statistical summary including:
  - Mean
  - Median
  - Standard Deviation
  - Min/Max
  - Variance

## ✅ Monthly Sales Growth Analysis
- Created monthly sales trend analysis.
- Added up/down arrows using Conditional Formatting.

## ✅ High Value Customer Identification
- Identified high-value customers using:
  - `FILTER()`
  - `INDEX()`
  - `MATCH()`

## ✅ Pivot Table Analysis
- Analyzed total sales by:
  - Region
  - Product Category

## ✅ Data Visualization
- Created:
  - Bar Chart
  - Line Chart
  - Pie Chart

## ✅ Interactive Dashboard
- Combined KPIs and charts into a professional dashboard.

# 📊 Key Insights

- West region generated strong sales performance.
- Higher discounts negatively impacted profit.
- A small group of customers contributed major revenue.
- Sales fluctuated across months showing seasonal variation.
- Electronics and Books categories showed high engagement.
